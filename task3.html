Task: CV Webpage Design Explanation

1. Did you use any W3 Schools templates or did you design the parallax on your own?

I did not use any W3 Schools template. I designed the parallax effect on my own using background-attachment: fixed and custom CSS styling(with help of generative AI).

2. Which sections did you put in your CV?

I included Profile, Education, Skills, and Contact sections in my CV website.

3. What colour combinations did you give? Anything more than the usual dark mode?

I used green gradient for left side and dark transparent content boxes on right side. I also added a Blue Theme option, not only normal dark mode.

4. Which interactive elements did you incorporate into your design?

I added theme change buttons, smooth scrolling navigation links, hover scaling effects, and floating animation for content boxes.

5. Briefly explain your Javascript code for interactive portions of the webpage.

My JavaScript functions add or remove a “blue” class to the body when buttons are clicked. This changes colours dynamically without reloading the page.

6. Did you use different coloured backgrounds to distinguish between the left and right parts? Which colours and why?

Yes, I used green gradient for left fixed panel and parallax image backgrounds for right sections. This makes both parts clearly different and visually attractive.
