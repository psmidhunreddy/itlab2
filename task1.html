1. What are the two main sections of an HTML file?

I used the <link> tag to link the external CSS file.

4. If you didn't separate them, what HTML tags did you use to specify the CSS content?

I used an external CSS file.

5. In which section of the HTML file did you put them?

The CSS file was linked in the <head> section of the HTML file.
